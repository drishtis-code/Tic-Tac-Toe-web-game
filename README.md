# 🎮 Tic Tac Toe

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features
- Two-player gameplay
- Winner and draw detection
- Restart game functionality
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## Run Locally
Open `index.html` in your browser to play the game.
